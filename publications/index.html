Publications

Longer-form public artefacts: whitepapers, frameworks, guides, reports, and other pieces that need more structure than a blog post.

Publishing model

Markdown remains the canonical source. The website provides readable HTML, while substantial publications can also offer a downloadable PDF once the approved content and final export are ready.

Only explicitly public material belongs here. If a source is still private, internal, or ambiguous, it should stay out of the site until reviewed.
